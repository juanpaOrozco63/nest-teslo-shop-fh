# Teslo API


1. Clonar Proyecto
2. Ejecutar ``` npm i ```
3. Clonar el archivo ``` .env.template ``` y renombra a ``` .env ```
4. Cambiar las variables de entorno
5. Levantar la base de datos con el siguiente comando

```
docker-compose up -d
```
6. Ejecutar SEED
```
localhost:3000/api/seed
```
7. Levantar el aplicativo 
``` 
 nest start
```
